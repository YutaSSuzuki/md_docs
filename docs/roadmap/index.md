# Roadmap

このディレクトリは、現在の開発状況と次にやることを把握するための資料を置く場所です。

ここには「今どこまで終わっているか」「次に何をするか」「何が保留か」を書く。

## What To Write

- 完了済みの作業
- 進行中の作業
- 次にやる作業
- 保留中の作業
- 判断待ちの作業
- マイルストーン

## Documents

| Document | Purpose |
| --- | --- |
| [Current Roadmap](current-roadmap.md) | 現在の開発状況と次の作業を把握する |

## Reading Flow

1. [Current Roadmap](current-roadmap.md) で現在位置を確認する。
2. 詳細な未実装案がある場合は `../proposals/` を確認する。
3. 実施済みの変更は `../changelog/` を確認する。

## AI Update Rule

AI が作業を完了した場合、必要に応じて `current-roadmap.md` の状態を更新する。

未実装の構想は Roadmap に詳細を書きすぎず、`../proposals/` に分ける。
